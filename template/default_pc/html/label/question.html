  • 常見問題

    提示TOKEN驗證失敗、提示登錄超時、提示向服務器請求數據失敗、或者其他大部分問題，請【重新登陸】，或者換個瀏覽器，推薦使用谷歌瀏覽器。

  • 無法支付
    • 請重新發起支付。
    • 換一個價格支付。
    • 使用自帶瀏覽器打開支付。
  • 支付後沒有成為VIP會員
    • 請【重新登錄】賬號。
    • 查看自己是否確實支付成功。
    • 聯繫在線客服為您處理問題。
  • 賬號已經被使用了，怎麼辦？

    您可能已經註冊過了，請至【忘記密碼】設定新密碼登錄。

  • 我忘記密碼了

    請連接至登錄專區【忘記密碼】申請。

  • 為什麼我的郵箱收不到修改密碼的通知？
    • 註冊時請填寫真實正確的郵箱，並檢查垃圾郵件。
    • 完成註冊後，建議先進行郵箱驗證，確保郵箱有效。手機可直接驗證。
    • 如果收不到郵件或手機驗證碼，請聯繫【在線客服】處理。
  • 成為VIP有什麼好處嗎？

    無限制的觀看所有線上影片，還可以將影片收藏喔！

  • 我該如何成為VIP？

    請至【VIP升級】專頁，進行會員升級吧！

  • 升級VIP的支付方式有哪些？

    提供微信/支付寶，來升級您的賬戶。

  • 我已經升級1個月VIP，想更換成年費VIP可以嗎？

    可以喔！一樣至【VIP升級】升級您的會員等級，將以原VIP期限延長12個月。

  • 我已經升級了，但是網頁依然要求升級才可以觀看

    可能是您瀏覽器的cookie失效，建議您先退出賬號，清除cookie再登錄就會成功。

  • 鑽石用途？

    鑽石可以作為儲值幣使用。可以兌換多種產品。

  • 鑽石使用方式？
    • 平台提供“稀缺資源+自產資源”，需要鑽石才能觀看，並且在扣點後再觀看無需扣點。
    • 若您是普通會員，請先充值「VIP」詳見【充值鑽石】
    • 若您已是「VIP」亦可單獨充值鑽石，請見【充值鑽石】。
  • 緩存點是什麼？

    當視頻訪問卡頓，或者您特別鍾愛某一部視頻。您可以使用緩存點。

  • 緩存點怎麼使用？

    點擊需要緩存的視頻，可緩存該視頻至您的APP上。

  • 注意事項

    一部視頻消耗緩存點後，以後使用此賬號，可以24小時內重複緩存該視頻。

  • 影片播放後沒有反應
    • 請確認您的網絡是否穩定，以及瀏覽器版本。
    • 聯繫【在線客服】為您處理。
  • 系統提示我等級不足，無法觀看影片

    該影片僅提供VIP觀看，請至【VIP升級】升級您的會員等級；若您已是VIP，請【登錄】。
